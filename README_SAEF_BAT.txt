SAEF Batch Launchers
=====================

File:
- run_app.bat                -> Jalankan app.py langsung (pakai .venv jika ada; kalau Streamlit belum ada, install otomatis)
- setup_venv_and_run.bat     -> Buat .venv, install requirements.txt, lalu jalankan app.py

Cara pakai:
1) Salin kedua .bat ini ke folder yang sama dengan app.py (dan requirements.txt kalau ada).
2) Double-click salah satu:
   - run_app.bat (cepat)
   - setup_venv_and_run.bat (disarankan: terisolasi di .venv)
